# 朝花

## 求索——2024.5

学习物理使我感到冰冷，因为会思考到人类悲哀的未来；但是，人就是在这种无意义中过来的，不对，我没有在想这种自得其乐。我是强制自己冷静吗？我是借助伟人的思考，想到人都会死亡吗？还是，我把这些事都留到了以后的自己呢？首先，我可能接受了自己的死亡，我不再想着永生，我接受个体的渺小和个人意志的短暂，由此把自己和人类族群做出了分离；其次对于人类，我也接受了人类的渺小与无力，在宇宙这个维度上，人类是极容易就此湮没的，人只能摸索着“走下去”，“走下去”！

我想要，意志；我想要，力量，坚持让我走下去的力量。走下去、走下去，终究这样的宣言是苍白无力的。。。。。。真的吗？你要否定过去走来的一切吗？分明是手段，为何成为了目的？那个人的解读，真的是对的吗？关于自己的求索，真的能这么简单地定性吗？支撑你走到现在的，真的不是身边陪伴的可爱的人吗？在那时，我抓住了一根救命稻草；现在，我需要重新思考那根稻草的含义，我不能永远依靠那根稻草，我需要寻找**我**的出路，而不是追随先生的路——何况也只是另一人解读的先生的路。

没想到没想到，我原来才是最崇拜偶像的那个人，分明我最讨厌偶像崇拜了。

我将继续思考我的出路、思考我的构成了，虽然未来更加地混沌了，但是这也是值得令人庆幸的，我摆脱了一种自我施加的束缚。当年编织了用于自我保护的茧，而今该破茧了，不再坚硬，反而幼嫩，正因幼嫩，才能接纳并再次求索。

求索什么？只是求索我要到哪里去这个“简单”的问题罢了。

如何求索？接着去接纳知识，但是在这期间，要去思索，知识的意义。没事，发现它没有意义，也比不去发现、浑噩地接受要强上百倍千倍！
