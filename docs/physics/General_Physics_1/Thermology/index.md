# 热学

!!! info
    热学内容有：热力学温标、热膨胀、热传导、理想气体状态方程、温度和压强的微观机制、平均自由程、Maxwell Speed Distribution、实际气体状态方程、热力学第一定律(就是等温等压绝热膨胀压缩之类的，涉及 p-v 图，这是热学的重难点)、热力学第二定律(熵)。
    热学总体概念、公式较多，比较繁琐，但题目难度不大，深度较浅。一般而言不需要额外阅读教材，看看 ppt 或者 google 一下足以(看本教材也足矣)。
